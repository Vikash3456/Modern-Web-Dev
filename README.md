## 💻 Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/yourusername/Modern-Web-Dev.git
```

2. Navigate to project directory
```bash
cd Modern-Web-Dev
```

3. Open `index.html` in your browser or use a live server
